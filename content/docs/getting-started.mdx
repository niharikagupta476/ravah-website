## 1) Connect CI/CD

Start by connecting GitHub, GitLab, or Jenkins to ingest pipeline runs and deployment metadata.

## 2) Configure incidents

Connect Slack and PagerDuty so Ravah can correlate incident timelines with delivery events.

## 3) Invite your team

Add responders, set alert preferences, and run your first guided RCA workflow.
